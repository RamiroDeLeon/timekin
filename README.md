# Timekin

A journaling app to preserve personal memories with text, media, location, and mood.

## Tech Stack

- React (Vite) + TypeScript
- Node.js + Express
- MongoDB (Atlas)
- AWS S3 (media)
- Tailwind CSS
- Terraform (infra)
- GitHub Actions (CI/CD)

## Getting Started

```bash
git clone git@github.com:RamiroDeLeon/timekin.git
cd timekin
npm install
